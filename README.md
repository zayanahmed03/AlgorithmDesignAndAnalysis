实验内容

一、排序算法实验

本实验实现并分析了以下三种经典排序算法：

* 冒泡排序（Bubble Sort）
* 归并排序（Merge Sort）
* 快速排序（Quick Sort）

实验内容包括：

1. 不同输入规模下的比较次数统计；
2. 输入等价类测试；
3. 归并排序与快速排序递归过程分析；
4. 算法性能比较与复杂度分析。



二、0-1背包问题实验

本实验分别采用以下四种算法求解0-1背包问题：

* 贪心法（Greedy Algorithm）
* 动态规划法（Dynamic Programming）
* 回溯法（Backtracking）
* 蛮力法（Brute Force）

实验内容包括：

1. 随机生成物品数据；
2. 不同物品数量和背包容量下的实验测试；
3. 运行时间与求解结果分析；
4. 不同算法性能比较。
